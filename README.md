# Portfolio

My creative software engineering and digital art portfolio. Built with React and Vite.

## Features

- **Digital Artistry Showcase**: Sleek bento-style skills matrix and selected works section.
- **Vibe Coding Inspired Design**: Ambient mouse parallax effects, glassmorphic UI cards, and smooth transitions.
- **Client-Side Protections**: Disables casual browser inspection (right-click, F12, DevTools shortcuts).
- **Security Audited**: Built-in XSS input sanitization and payload limits.
- **Vite & React**: Fast compilation and Hot Module Replacement (HMR).
